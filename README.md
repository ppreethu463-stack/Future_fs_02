Project Description
A simple Customer Relationship Management (CRM) system to manage leads, track status, and improve client communication.

🚀 Features
Add and manage leads
Track lead status (New, Contacted, Follow-up, Closed)
Dashboard with statistics
Pipeline board view
User login system
